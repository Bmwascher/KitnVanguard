---
description: Scaffold a new addon feature module with interview
---

Interview me about the new module using these questions:
1. What should the module be named and what does it do?
2. Does it need slash commands? If yes, what commands and what do they do?
3. Does it need SavedVariables (persistent settings)?
4. What WoW events does it need to listen to?
5. Does it interact with any combat APIs (health, auras, spells)?
6. Does it create any UI frames?

After the interview, use the module-scaffolder agent to create the module.
Then run /api-validate to verify the new code is 12.0 compliant.
